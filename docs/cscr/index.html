About the Project

REPH Mealeys

MS to Ops Macro (v1.2)

This project is an enhancement of my self-written macros that I have developed for production use. Ultimately, it aims to address the following user stories that I have formulated as the project owner:

As the project owner, I want this macro to improve so that the script would be stable and be more useful.

As the project owner, I want this macro coded in an icon so that it can have a more user-friendly interface.

As the project owner, I want this icon incorporated in the Mealeys existing Tab for convenience to launch it any time.

Thanks, and enjoy!
